## Compilation/Installation Guide

To install the module `acse4-4-likelihood
` clone the respository to your computer with 
``
git clone https://github.com/acse-2019/acse4-4-likelihood.git
``
Then change directory into the root of the repo you just cloned 
``
cd cse4-4-likelihood
``
